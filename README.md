# Random_Problems
##Problem_1
  print_steps.py contains the solution to the following question: 
  given N number of steps and the height of each step, print the 2D staircase that would result. (ASSUME ALL STEPS TO HAVE WIDTH=HEIGHT)

EXAMPLE 1:

  Input:
  4 
  1
  2
  1
  3

  Expected Output:

            * * * * 
            *
            *
          * * 
      * * * 
      *
    * * 
    *

EXAMPLE 2:

  Input:
  2
  1
  1
  Expected Output:
  
      * * 
    * * 
    *
